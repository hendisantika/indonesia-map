# Screenshot Instructions for Next.js Interactive Map

Please take the following screenshots and save them in the `img/` directory:

## 1. Loading State (nextjs-loading.png)
- Navigate to http://localhost:3000/interactive
- Capture the page while the map is loading (with the spinner visible)
- Show the disabled province selector with "(Loading map...)" text

## 2. Provinsi Boundary (nextjs-provinsi.png)
- Select a province (e.g., "32 - JAWA BARAT")
- Capture the map showing the blue province boundary polygon
- Show the province details panel on the left
- Ensure the map is zoomed to show the province boundary

## 3. Kabupaten Boundary (nextjs-kabupaten.png)
- Keep the province selected, then select a kabupaten (e.g., "32.04 - KABUPATEN BANDUNG")
- Capture the map showing the kabupaten boundary
- Show the selected kabupaten in the dropdown and details panel

## 4. Kecamatan Boundary (nextjs-kecamatan.png)
- Select a kecamatan (e.g., "32.04.28 - Bojongsoang")
- Capture the map with the kecamatan boundary displayed
- Show all three dropdowns filled and the detail panel
- Ensure the map is zoomed to the kecamatan area

## 5. Desa Boundary (nextjs-desa.png)
- Select a desa/kelurahan (e.g., "32.04.28.0002 - Bojongsoang")
- Capture the map with the small desa boundary polygon
- Show all four dropdowns filled and complete details
- Map should be tightly zoomed to the village area

## Screenshot Tips:
- Use browser's full-screen mode (F11) for clean captures
- Include both the left sidebar and the map area
- Make sure boundaries are clearly visible
- Capture at a resolution of at least 1920x1080
- Save as PNG format
